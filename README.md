# OrbitScale Insurance | Protection DNA Matrix

OrbitScale is a specialized insurance advocacy platform designed to provide technical trust, data-driven transparency, and seamless claim advocacy for modern families.

## 🛡️ Core Features
- **Protection DNA Builder**: Interactive risk profiling and coverage matrix.
- **Claims Advocacy Trail**: Transparent settlement timeline from incident to 100% cashless resolution.
- **Regulatory Vault**: High-trust compliance markers and live settlement ratio monitoring.
- **Layered Protection**: Synergistic coverage logic (Base Core + Specialized Boosters).

## 🛠️ Technical Stack
- **Frontend**: React, Vite, TypeScript
- **Styling**: Tailwind CSS, Framer Motion
- **Icons**: Lucide React
- **Verification**: IRDAI Compliance Logic

## 🚀 Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
